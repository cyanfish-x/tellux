## 文档定位

Tellux 文档面向库使用者，提供从初始化 Viewer 到加载真实地理数据的完整路径。

- **指南** 解释常见工作流和推荐写法。
- **API 文档** 记录公开 TypeScript API 的参数、返回值和生命周期。
- **能力参考** 汇总底层依赖能力，帮助判断 Tellux 当前封装到什么程度。
