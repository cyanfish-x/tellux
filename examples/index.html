Tellux Examples

这些示例从本仓库的 src 直接引入 Tellux，适合开发时快速检查基础场景和 Viewer 事件。

基础场景渲染 创建 Viewer，通过 TemplateUrlResource 加载 ArcGIS 影像瓦片，并设置初始相机视角。 相机飞行 使用 Cesium 风格的 viewer.flyTo 在多个城市视角之间平滑飞行。 点击事件 监听 Viewer canvas 点击事件，并显示点击位置的像素坐标。 数据源切换 通过按钮切换 Cesium Ion 和 XYZ 模板影像来源。 Cesium 地形热切换 输入 quantized-mesh 地形 URL，通过左上角面板开启、关闭并热切换地形。
