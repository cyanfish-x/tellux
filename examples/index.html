示例

示例入口

这些页面通过 Vite 从仓库源码直接引入 Tellux，适合开发时验证 API 行为和渲染效果。

查看器 → 基础场景渲染 创建查看器，加载 ArcGIS 影像瓦片，启用默认大气、云层和后处理，并提供东京、上海视角按钮。 相机 → 相机飞行 使用 Cesium 风格的相机飞行 API 在东京、上海、新加坡之间平滑飞行，并支持取消飞行。 事件 → 点击与鼠标读数 监听查看器点击和鼠标移动事件，显示画布像素坐标以及经度、纬度和高度。 图层 → 影像图层管理器 管理 ArcGIS XYZ、NASA GIBS WMS 和 OpenInfraMap MVT 图层，支持显隐、透明度和拖拽排序。 三维瓦片 → 三维瓦片加载 通过 tileset.json 地址或 Cesium Ion 资产加载独立场景三维瓦片，并支持飞行定位、显隐和移除。 地形 → Cesium 地形热切换 输入量化网格地形地址，通过左上角面板开启、关闭并热切换地形。 大气 → 体积云与大气 切换太平洋和喜马拉雅视角，并通过公共设置面板调整大气、体积云、UTC 时间、光照和曝光。
