Tellux Examples

这些示例从本仓库的 src 直接引入 Tellux，覆盖基础场景、相机控制、事件读数、影像图层、3D Tiles、地形和大气云层。

基础场景渲染 创建 Viewer，加载 ArcGIS 影像瓦片，启用默认大气、云层和后处理，并提供东京、上海视角按钮。 相机飞行 使用 Cesium 风格的 viewer.camera.flyTo 在东京、上海、新加坡之间平滑飞行，并支持取消飞行。 点击与鼠标读数 监听 Viewer 点击和鼠标移动事件，显示 canvas 像素坐标以及经度、纬度和高度。 影像图层管理器 管理 ArcGIS XYZ、NASA GIBS WMS 和 OpenInfraMap MVT 图层，支持显隐、透明度和拖拽排序。 3D Tiles 加载 通过 tileset.json URL 或 Cesium Ion asset 加载独立场景 3D Tiles，并支持飞行定位、显隐和移除。 Cesium 地形热切换 输入 quantized-mesh 地形 URL，通过左上角面板开启、关闭并热切换地形。 体积云与大气 切换太平洋和喜马拉雅视角，并通过公共设置面板调整大气、体积云、UTC 时间、光照和曝光。
